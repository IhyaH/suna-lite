# GitHub 仓库描述

## 简短描述
Suna-Lite - 轻量级AI Agent系统，基于OpenAI API构建，集成文件操作、命令行执行、Web搜索和浏览器自动化功能

## 详细描述
Suna-Lite 是一个现代化的轻量级AI Agent系统，基于OpenAI API构建，集成了文件操作、命令行执行、Web搜索和浏览器自动化等核心功能。采用异步编程架构，提供安全可靠的智能助手体验。

### 主要特性
- 🤖 智能对话：基于OpenAI GPT模型的自然语言交互
- 📁 文件操作：完整的文件系统管理功能
- 🖥️ 命令执行：安全的Shell命令执行环境
- 🌐 Web搜索：使用Tavily API获取实时网络信息
- 🌍 浏览器自动化：网页交互和数据提取
- 🏗️ 工作区管理：隔离的任务执行环境
- 💅 美观界面：基于Rich库的彩色CLI界面
- ⚙️ 高度可配置：灵活的YAML配置系统

### 技术特点
- **异步架构**：高性能的异步编程模式
- **模块化设计**：工具系统可扩展，支持动态注册
- **安全机制**：路径隔离、命令过滤、沙箱环境
- **配置管理**：简化的YAML配置系统，无需环境变量文件

### 适用场景
- 自动化文件管理和处理
- 智能命令行操作
- 网络信息搜索和整理
- 网页数据抓取和分析
- 开发辅助和代码生成
- 系统管理和监控
